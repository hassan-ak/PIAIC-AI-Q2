# Terminal Shortcuts

| Sr. No. | ShortCut         | Description          |
| ------- | ---------------- | -------------------- |
| 1       | TAB              | Autocomplete Words   |
| 2       | Arrow Up or Down | Show old commands    |
| 3       | Ctrl A           | Move curser to start |
| 4       | Ctrl E           | Move curser to end   |
| 5       | Ctrl C           | End command          |
